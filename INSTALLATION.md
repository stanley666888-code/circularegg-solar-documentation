# Installation Guide

## Prerequisites

Before installation, ensure you have:

- [ ] Basic understanding of electrical systems
- [ ] Required tools (see below)
- [ ] Safety equipment (gloves, safety glasses)
- [ ] Clear access to installation site
- [ ] All components and documentation

## Tools Required

- [ ] Ladder or lifting equipment
- [ ] Wrench set (adjustable recommended)
- [ ] Screwdrivers (Phillips and flathead)
- [ ] Hammer
- [ ] Level
- [ ] Measuring tape
- [ ] Cable connectors
- [ ] Weatherproof sealant

## Site Preparation

### 1. Select Installation Site

**Requirements**:
- Flat, stable ground
- Unobstructed solar exposure
- At least 5 meters from electrical sources
- Accessible for maintenance

**For Vehicle-Mounted**:
- Vehicle capable of supporting system weight
- Clear area for deployment
- Proper towing or towing accessories

### 2. Prepare Ground Surface

**For Fixed Installation**:
- Clear debris and rocks
- Compact the ground if needed
- Level the surface

**For Mounted Installation**:
- Check vehicle tire pressure
- Ensure vehicle is on level ground

## Installation Steps

### Step 1: Unpack and Inspect

1. **Remove from Packaging**
   - Carefully unpack all components
   - Inspect for any damage during transport

2. **Verify Components**
   - Check light tower
   - Verify solar panels
   - Confirm battery and wiring
   - Ensure all mounting hardware included

3. **Check Batteries**
   - Verify batteries are fully charged
   - Check for any leaks or damage

### Step 2: Position System

**Fixed Installation**:
1. Position the light tower in desired location
2. Ensure panels face south (northern hemisphere) or north (southern hemisphere)
3. Allow for 2-3 meters clearance on all sides for operation

**Vehicle-Mounted**:
1. Park vehicle in suitable location
2. Ensure vehicle is in park with parking brake engaged
3. Position vehicle to maximize solar exposure

### Step 3: Mount Solar Panels

1. **Mount Panels**
   - Attach solar panels to designated mounting points
   - Ensure panels are securely fastened
   - Verify panel tilt angle (typically 20-30 degrees)

2. **Secure Wiring**
   - Connect panels to battery/inverter
   - Use weatherproof connections
   - Secure wires to prevent movement

### Step 4: Connect Battery

1. **Connect Battery**
   - Attach battery to charging system
   - Ensure proper polarity (positive/negative)
   - Tighten all connections

2. **Power On**
   - Turn on main power switch
   - Check battery indicator lights

### Step 5: Test System

1. **Basic Functions Test**
   - Turn lights on/off
   - Test rotation mechanism
   - Verify all lights function
   - Check charge indicator

2. **Extended Test**
   - Run lights for 4-6 hours
   - Monitor battery levels
   - Test rotation during operation

### Step 6: Final Adjustments

1. **Adjust Illumination**
   - Set light angle for optimal coverage
   - Position tower for best 360° coverage

2. **Secure System**
   - Ensure all components are properly secured
   - Check for any loose parts

## Safety Precautions

⚠️ **CRITICAL SAFETY RULES**

1. **Electrical Safety**
   - Never work with live electrical systems
   - Disconnect power before maintenance
   - Use appropriate personal protective equipment

2. **Working Heights**
   - Use proper ladders or equipment
   - Never stand on unstable surfaces
   - Have safety harness if working above 3 meters

3. **Weather Conditions**
   - Do not install during high winds (>15 km/h)
   - Avoid extreme temperatures (>40°C or <-10°C)
   - Do not install during heavy rain

4. **Overhead Hazards**
   - Check for overhead power lines
   - Maintain safe distance from electrical sources
   - Identify clearance requirements

5. **Heavy Lifting**
   - Get assistance for heavy components
   - Use proper lifting techniques
   - Avoid twisting motions

## Common Installation Mistakes

❌ **Mistakes to Avoid**:

1. **Incorrect Solar Orientation**
   - Panels facing wrong direction
   - **Fix**: Check compass orientation

2. **Poor Ground Preparation**
   - Uneven or unstable surface
   - **Fix**: Level and compact ground

3. **Loose Connections**
   - Wires not properly secured
   - **Fix**: Tighten all connections

4. **Neglecting Safety**
   - No safety equipment
   - **Fix**: Always wear PPE

5. **Rushing Installation**
   - Skipping test procedures
   - **Fix**: Follow all steps methodically
